Principais Formatações

Negrito ou Destaque

É posspivel usar um termo em negrito usando a tag b

É possível também usar um termo em destaque usando a tag strong

Itálico ou Ênfase

É possível usar o Itálico usando a tag i

É possível também utilizar o ctrl + shit + p para selecionar um texto e usar o wrap with abreviaton para usar o Itálico

A tag mais indicada para Itálico é a em

Texto Marcado

Um texto pode ser marcado com a Tag Mark

Texto Grande e Texto Pequeno

Textos Grandes e Textos pequenos podem ser misturados no mesmo parágrafo

Texto Deletado

Podemos marcar um texto que pode ser lido, mas não precisa ser considerado

Texto Iserido

Marca-se um texto para indicar que foi adcionado posteriormente

Texto Sobescrito

Texto para ser utilizado no alto das palavras, por exemplo: x3 - 2x2 + 5x = 15

Texto Subescrito

Texto para ser usado na parte inferior das linhas, por exemplo n1, n2, n3...
